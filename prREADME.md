## 📝 Описание программы

**Weather Diary** — это простое графическое приложение для ведения дневника наблюдений за погодой.

### Основные возможности:

- ✅ **Добавление записей** — дата, температура, текстовое описание, осадки (да/нет)
- ✅ **Просмотр записей** — все записи отображаются в удобной таблице
- ✅ **Фильтрация**:
  - По дате (показать записи за конкретный день)
  - По температуре (показать записи выше заданного значения)
  - Комбинированная фильтрация (по двум критериям сразу)
- ✅ **Удаление записей** — удаление выбранной записи из таблицы
- ✅ **Автосохранение** — все данные сохраняются в файл `weather_data.json`
- ✅ **Автозагрузка** — при запуске программа загружает сохранённые данные
- ✅ **Валидация ввода**:
  - Дата проверяется на формат `ГГГГ-ММ-ДД`
  - Температура должна быть числом
  - Описание не может быть пустым
