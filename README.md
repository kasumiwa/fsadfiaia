## Описание программы 
Random Quote Generator — это графическое приложение для генерации вдохновляющих цитат. Позволяет получать случайные цитаты известных людей, добавлять собственные цитаты с проверкой на пустые поля, просматривать историю сгенерированных цитат с фильтрацией по автору и теме, а также сохранять и загружать историю в JSON-файл."
## Требования для запуска 
Что нужно установить на компьютер: - Python 3.10 или выше - Библиотеки: `pip install matplotlib requests` ---
